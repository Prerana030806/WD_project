# LearnoHub 🚀

A beginner-friendly programming learning website built with pure HTML, CSS, and JavaScript — no frameworks, no build tools, no complexity.

---

## 🌟 Live Features

- 4 Courses — Web Development, Python, Data Structures, Git & GitHub  
- 17 Topics — each with clear explanation, real code examples, and a pro tip  
- Quizzes — 5 MCQ questions per course with score saving and full answer review  
- Login — name + email stored in localStorage, no server needed  
- Responsive — works on mobile and desktop  

---

## 📁 File Structure

learnohub/
└── index.html   ← entire app (HTML + CSS + JS in one file)

That's it. One file.

---

## ▶️ How to Run

Just open index.html in any browser. No installation needed.

If using the Replit dev server:

pnpm --filter @workspace/learnohub run dev

Then visit the preview URL.
